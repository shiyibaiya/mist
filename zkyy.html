各个题型详解：

一、听力：

1. 听之前要先浏览题目，通过题目可以适当猜测听力内容，做到心中有数

2. 划出题目和选项中的关键词，帮助快速选出正确答案

3. 注意问题中所指向的人物

4. 时间和数字要特别仔细听，并适当做笔记，注意区分几十（-ty）和十儿（-teen）。当对话中出现几 个时间，前而所提到的时间往往都是不对的，有时可能会需要进行简单的计算，要适当做笔记：当涉及数目较大时，要注意英语中数字是三位一起读的，书写是每三位一个逗号

5. 注意区分特殊疑问词和时间状语

阅读理解（CD 两篇短文）

审题：找出题干或选项的关键词（人名，地名，时间······）

阅读全文，确定该题在文章中的位置

对照题干和选项进行分析选择，排除干扰项

注意做题时要标记好信息点和题号，便于检查。

问题中常用单词

passage／article／text 文章        paragraph     段落      writer/author  作者

underlined 下划线              following  以下的       above  以上

true 真实的                   wrong 错误的         except 除了

purpose 目的                   meaning 意思         main idea     主旨大意

best title 最好的标题            infer 推断

Introduce a topic／theme 引出问题

give a example 举例子

provide a solution 提出解决方法

细节理解题技巧：

1.绝对化的描述要注意，一般不是正确答案

2.选项中带有时间，数字，关系对比（比较级或最高级），范围，否定表述 都可能是设置干扰项的地

方，需要仔细和文章进行对比理解

3.正确选项的概括性比较强，片面的，表面的一般不是正确答案

4.两个选项意思截然相反，一般有一个是正确答案

词义猜测题技巧；

1．仔细阅读这个需要猜测的词或短语的前后句，找到上下文所给的提示

如：破折号后面的内容：

文中进一步给出的解释：

同位语或并列关系的意思：

逻辑连词的提示（因果／转折／让步···）；

构词法等

 


 


2．把猜测的词义带入文章中再仔细阅读一遍检查

主旨题技巧：

Title 题／purpose 目的／main idea／mainly about 主旨···

1．快速阅读文章首段（有时在第1,2段），尾段的首尾句，注意概括性强的句子，以及是否存在转折词．2．了解文章大致结构和脉络

3．分辨不清的选项可以尝试用反推法

主旨题错误答案的特点：

1. 以偏概全

2.过于笼统

3.没有体现文章观点

4. 无中生有

推理判断题

1．做这类题目时，要首先在文章中找出据以推断的有关文字，然后加以分析，尤其要领悟隐含在字里行 间的内涵，体会作者的言外之意、弦外之音。

2．推理判断时一定要以原文内容为前提，从作者的观点去思考，切勿凭个人观点、习惯看法来选择答案3．善用排除法排除干扰选项。

 

五选五

1.答题前先扫一遍全文结构（是否是小标题式，总分总式······）

2. 作答前先细读每个选项（重点注意每句的关键词，转折词，代词，大小写，句式特点）3．答题时按照先易后难原则，注意所填句子与上下文的关联性。

4．作答完成后，再次通读全文

小技巧：

1.首段尾句设空一般为承上启下

2.小标题设空处句子结构一般为祈使句（且为本段主旨）

3.注意空格前后和选项中的代词指代

4.关键词的复现或同义词复现

5.大小写也可以是解题突破点

6.顺序词和标点符号

7. 前后句的句式结构

 


 


完型填空（时间分配10-12 分钟，如果特别难，最多不超过15分钟）

1.完型填空一般为故事说理或者是说明文说理，首段很重要，必须仔细阅读

2.选择的选项通常在文章上下文有提示，仔细阅读来进行选择

3. 多留心文章中作者使用的表示情感态度的形容词，副词，动词以及转折词

4. 同元素排除

5. 选项平均化

4.形容词，副词：

形容词修饰名词或代词，跟在be动词或感官动词后而

副词修饰其余的（动词／副词／形容词／句子）

形容词变副词的规则

1）直接加ly             careful--carefully

2）辅音字母加y 结尾的，改y为i，加ly          happy--happily

3）le 结尾的，去e加y          simple--simply

4）ue 结尾的去e 加ly       true--truly（初中阶段只有这一个）

5）II结尾的，直接加y       full---fully

 


 


6）有些单词，既可以作形容词，也可以作副词fast, early, late, hard

5.动词：谓语动词（时态，语态）

时态：

1）一般现在时      主语＋V原     主语（三单）

2）一般过去时      主

3）一般将来时      主语 原   主语＋be（am／is／are）going to＋V 原

4）现在进行时        

5）过去进行时      主语

6）现在完成时      主语＋have／has＋过去分词

注意：现在完成时和一般过去时的用法区分

一般过去时有具体的过去时间，如 yesterday，the day before yesterday， last， ago， in the past，in＋过去年份

现在完成时是表示过去的动作对现在有影响或动作从过去延续到现在，标志词有 since，for，so far，in the last／past...years， already， yet 等

 

被动：

1）一般现在时 am/is/are+done

2）一般过去时 was/were+done

3）一般将来时 will +be done

4）现在完成时   have/has been done

5）情态动词     情态动词＋be done
